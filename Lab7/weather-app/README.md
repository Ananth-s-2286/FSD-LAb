# Crazy Weather App

A fun and crazy weather web app that fetches real-time weather data using the Open-Meteo API.

## Features

- Search for weather by city name
- Use current location for weather
- Displays temperature, weather description, wind speed, and humidity
- Weather icons and symbols
- Crazy animations: floating container and rain effect when it's raining

## How to Run

1. Open `index.html` in your web browser.
2. Enter a city name and click "Search" or click "Use Current Location".
3. Enjoy the crazy weather display!

## APIs Used

- [Open-Meteo Geocoding API](https://geocoding-api.open-meteo.com/) for city search
- [Open-Meteo Weather API](https://open-meteo.com/) for weather data

No API key required!

## Technologies

- HTML5
- CSS3 (with animations)
- JavaScript (ES6+)

## Troubleshooting

- If the app doesn't load data, check your internet connection.
- For current location, allow location access in your browser.
- If CORS issues occur, try running a local server (e.g., `python -m http.server` in the directory).
